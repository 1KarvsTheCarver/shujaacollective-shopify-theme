# Something's Better Than Nothing - Store Setup Guide

## Brand Overview
**Brand Name:** Something's Better Than Nothing
**Brand Message:** Motivational/Inspirational Apparel
**Target Audience:** People seeking positive, encouraging messages

## Brand Colors

### Primary Color Palette
- **Sunset Orange:** `#E8915B` - Main accent color
- **Deep Orange:** `#D67D47` - Primary button background
- **Burnt Orange:** `#C86A37` - Hover states
- **Cream/Beige:** `#F5E6D3` - Light backgrounds

- **Dark Teal:** `#2D4A4A` - Primary text and dark elements

## 1. BRANDING CONFIGURATION

### Logo Upload
**In Shopify Admin:**
1. Go to **Online Store → Themes → Customize**
2. Click **Theme settings** (usually in the left sidebar)
3. Navigate to **Logo**
4. Upload your logo files:
   - **Primary Logo:** Use the circular badge logo (retro sunset/mountain) or starburst logo
   - **Logo Height (Desktop):** 50-60px recommended
   - **Logo Height (Mobile):** 36-40px recommended

**Logo File Locations:**
- Circular badge logo (for headers with light backgrounds)
- Starburst logo (for feature sections)
- Simple text version (for minimal layouts)

## 2. COLOR SCHEME SETUP

### Creating Custom Brand Color Scheme

**In Shopify Theme Customizer:**
1. Go to **Theme settings → Colors**
2. Create a new color scheme called "Brand - Light"

**Brand - Light Scheme:**
```
Background: #F5E6D3 (Cream)
Headings: #2D4A4A (Dark Teal)
Body Text: #1F3838 (Deep Teal)
Primary Links: #D67D47 (Deep Orange)
Primary Button Background: #D67D47 (Deep Orange)
Primary Button Text: #F5E6D3 (Cream)
Primary Button Hover: #C86A37 (Burnt Orange)
Secondary Button Background: Transparent
Secondary Button Text: #2D4A4A (Dark Teal)
Secondary Button Border: #2D4A4A (Dark Teal)
Borders: #E8915B with 20% opacity
```

3. Create another scheme called "Brand - Dark"

**Brand - Dark Scheme:**
```
Background: #2D4A4A (Dark Teal)
Headings: #F5E6D3 (Cream)
Body Text: #EDD9BB (Light Cream)

Primary Button Background: #E8915B (Sunset Orange)
Primary Button Text: #2D4A4A (Dark Teal)
Primary Button Hover: #D67D47 (Deep Orange)
Secondary Button Background: Transparent
Secondary Button Text: #F5E6D3 (Cream)
Secondary Button Border: #F5E6D3 (Cream)
Borders: #F5E6D3 with 20% opacity
```

### Apply Color Schemes
- **Header:** Brand - Light or White
- **Hero Section:** Brand - Dark (with overlay on images)
- **Product sections:** Brand - Light

## 3. TYPOGRAPHY SETTINGS

### Recommended Font Pairings

**Option 1: Modern & Bold**
- Headings: **Montserrat** (Bold/700)
- Subheadings: **Montserrat** (Medium/500)
- Body: **Open Sans** (Regular/400)
- Accent: **Montserrat** (Bold/700)

**Option 2: Vintage Feel (matches retro sunset logo)**
- Headings: **Bebas Neue** or **Oswald**
- Subheadings: **Roboto Condensed**
- Body: **Lato** or **Raleway**
- Accent: **Bebas Neue**

**In Theme Settings → Typography:**
1. Set your chosen fonts
2. Adjust sizes:
   - H1: 48-56px (Desktop)
   - H2: 36-48px
   - H3: 28-32px
   - Body: 16px
   - Small text: 14px

## 7. SHOPIFY COLLECTIONS SETUP

### Create These Collections:

1. **All Products**
   - Type: Automated
   - Conditions: All products

2. **New Arrivals**
   - Type: Automated
   - Conditions: Product tag equals "new" OR Created date is within last 30 days

3. **Best Sellers**
   - Type: Manual
   - Add your top-selling products manually

4. **Hats & Caps**
   - Type: Automated
   - Conditions: Product type equals "Hat" OR Product tag contains "hat"

5. **T-Shirts**
   - Type: Automated
   - Conditions: Product type equals "T-Shirt" OR Product tag contains "tshirt"

6. **Hoodies**
   - Type: Automated
   - Conditions: Product type equals "Hoodie" OR Product tag contains "hoodie"

7. **Sale**
   - Type: Automated
   - Conditions: Compare at price is greater than Price

## 8. PRODUCT PHOTOGRAPHY GUIDELINES

### Photo Requirements:
- **Main Image:** Product on white or cream background
- **Image 2:** Product being worn (lifestyle shot)
- **Image 3:** Close-up of logo/design details
- **Image 4:** Product from different angle
- **Image 5:** Size/fit guide visual (for apparel)

### Image Specs:
- **Minimum Size:** 2048 x 2048 pixels
- **Recommended Size:** 2500 x 2500 pixels
- **Format:** JPG (for photos), PNG (for graphics with transparency)
- **File Size:** Under 500KB per image (optimize for web)

## MARKETING LAUNCH PLAN

### Pre-Launch (2 weeks before):
1. Create social media accounts (Instagram, TikTok, Facebook)
2. Post teaser content with logo and brand message
3. Build email list with landing page
4. Create launch discount code (e.g., LAUNCH20 for 20% off)

### Launch Week:
1. Announce launch on all social channels
2. Send email to subscribers
3. Run Instagram/Facebook ads
4. Partner with micro-influencers
5. Post daily content showing products

### Post-Launch:
1. Collect customer reviews
2. Share user-generated content
3. Run retargeting ads
4. Build email automation flows
5. Create blog content around brand values
